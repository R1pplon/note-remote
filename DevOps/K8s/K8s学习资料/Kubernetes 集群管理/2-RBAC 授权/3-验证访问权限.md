# 验证访问权限

我们该如何测试呢？我们不需要注销并以 Jane 的身份重新登录。Kubernetes 允许我们使用 `--as` 标志来「模拟」用户，从而测试权限。

这是一种实用的管理技术。当你能在将访问权限交给真实用户或自动化账户之前模拟身份时，权限检查会安全得多。它能帮助你在不产生不必要副作用的情况下验证意图。

让我们验证 Jane 是否可以列出 Pod：

```bash
kubectl auth can-i list pods --as jane
```

输出应该是 `yes`。

你还可以检查她是否可以获取特定 Pod 的详细信息：

```bash
kubectl auth can-i get pods --as jane
```

输出也应该是 `yes`。

这证实了 RoleBinding 正在生效。门禁卡在读卡器上可以使用了！
