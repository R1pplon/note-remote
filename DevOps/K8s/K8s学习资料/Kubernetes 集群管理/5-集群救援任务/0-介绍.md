# 简介

你是某快速发展的科技初创公司新上任的站点可靠性工程师（SRE）。现在是周五下午 4:30，首席工程师刚刚开启了为期一个月的假期。在离开之前，他们将新的「后端」微服务部署到了 `rescue-mission` 命名空间中，但监控仪表板随即变红。

部署失败了，服务无法路由流量，扩容触发器也卡住了。产品经理要求在 20 分钟内获得状态更新。

你的任务是深入集群，找出这些故障的根本原因，并将服务恢复到完全健康的状态。你需要修复崩溃的 Pod，修复服务发现，并确保集群允许应用程序扩容以应对周末的流量。

祝你好运，工程师。系统状态掌握在你的手中。
