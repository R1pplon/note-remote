# 部署应用程序

你的第一个任务是将应用程序清单部署到集群中。前任工程师将配置文件留在了 `/home/labex/project/rescue-mission` 目录中。

## 任务

1. 进入 `/home/labex/project/rescue-mission` 目录。
2. 将所有 YAML 配置文件应用到集群中。

## 要求

- Kubernetes 对象（Deployment 和 Service）必须在 `rescue-mission` 命名空间中创建（YAML 文件已指定此命名空间）。

## 示例

验证部署：

```bash
kubectl get all -n rescue-mission
```
