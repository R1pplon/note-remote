# 创建带有标签的节点

在此步骤中，我们将创建一个带有标签的节点，该标签将用于调度 Pod。

1. 在 `/home/labex/project` 目录下创建一个名为 `node-with-label.yaml` 的文件，内容如下：

```yml
apiVersion: v1
kind: Node
metadata:
  name: minikube
  labels:
    type: web
```

2. 应用更改：

```bash
kubectl apply -f node-with-label.yaml
```

3. 验证节点是否已创建并打上标签：

```bash
kubectl get nodes --show-labels
```
