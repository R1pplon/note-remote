# 创建命名空间

在这一步骤中，你将创建一个名为 `webapp` 的命名空间，以将 Web 应用程序与集群中的其他资源隔离。

创建一个名为 `namespace.yaml` 的文件，内容如下：

```yml
apiVersion: v1
kind: Namespace
metadata:
  name: webapp
```

使用以下命令将命名空间应用到你的集群中：

```shell
kubectl apply -f namespace.yaml
```

使用以下命令验证命名空间是否已成功创建：

```shell
kubectl get namespaces
```

你应该能在命名空间列表中看到 `webapp` 命名空间。
