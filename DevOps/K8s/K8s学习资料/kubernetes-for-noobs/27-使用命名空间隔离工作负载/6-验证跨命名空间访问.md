# 验证跨命名空间访问

首先，通过运行以下命令找到运行你应用程序的 Pod 的名称：

```shell
kubectl get pods -l app=other
```

你应该会看到 `other` Pod。记下 Pod 的名称。

接下来，运行以下命令，在运行你应用程序的容器中打开一个 shell 会话：

```shell
kubectl exec -it pod-name -- /bin/sh
```

将 `<pod-name>` 替换为你之前记下的 Pod 的名称。

进入 shell 会话后，运行以下命令来访问 `web-app` Deployment：

```shell
curl web-app.webapp
```

你应该会看到来自 Nginx Web 服务器的 HTML 响应。
