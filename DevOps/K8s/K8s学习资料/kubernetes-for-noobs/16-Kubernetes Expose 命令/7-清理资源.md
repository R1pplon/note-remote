# 清理资源

完成实验后，清理你创建的资源非常重要，以避免 Kubernetes 集群中不必要的资源消耗。

1. 通过运行以下命令删除 `hello-service`：

   ```bash
   kubectl delete service hello-service
   ```

2. 使用以下命令删除 `hello-world` 部署：

   ```bash
   kubectl delete deployment hello-world
   ```

清理资源可确保你的集群保持干净状态，为未来的实验做好准备。
