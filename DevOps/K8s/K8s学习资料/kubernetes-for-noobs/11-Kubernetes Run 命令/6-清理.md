# 清理

为了保持集群的整洁，请删除在实验期间创建的资源。

1. **删除资源**：

   运行以下命令：

   ```bash
   kubectl delete pod nginx-pod
   kubectl delete pod busybox-job
   kubectl delete deployment nginx-deployment
   ```

2. **验证清理**：

   检查是否没有剩余资源：

   ```bash
   kubectl get pods
   kubectl get deployments
   ```

   - 确保输出中未列出你创建的资源。
