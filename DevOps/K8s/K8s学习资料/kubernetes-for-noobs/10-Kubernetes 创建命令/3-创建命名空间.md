# 创建命名空间

命名空间（Namespace）允许你在 Kubernetes 中组织和隔离资源。

1. **创建命名空间定义文件**：

   打开一个名为 `namespace.yaml` 的新文件：

   ```bash
   nano namespace.yaml
   ```

2. **定义命名空间**：

   添加以下内容：

   ```yml
   apiVersion: v1
   kind: Namespace
   metadata:
     name: mynamespace
   ```

   按 `Ctrl+X`，然后按 `Y`，最后按 `Enter` 保存文件。

3. **应用命名空间**：

   创建命名空间：

   ```bash
   kubectl create -f namespace.yaml
   ```

4. **验证命名空间**：

   检查命名空间列表：

   ```bash
   kubectl get namespaces
   ```

   确认输出中显示 `mynamespace`。
