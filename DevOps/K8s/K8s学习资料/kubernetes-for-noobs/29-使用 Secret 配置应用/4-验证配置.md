# 验证配置

在这一步，你将验证你的应用程序是否已使用来自 `my-secret` Secret 的数据库密码正确配置。

通过运行以下命令找到运行你的应用程序的 Pod 的名称：

```bash
kubectl get pods -l app=my-app
```

你应该看到一个运行你的应用程序的 Pod。记下 Pod 的名称。

接下来，运行以下命令，在运行你的应用程序的容器中打开一个 shell 会话：

```bash
kubectl exec -it pod-name -- /bin/sh
```

将 `<pod-name>` 替换为你之前记下的 Pod 的实际名称。

进入 shell 会话后，运行以下命令以打印 `DATABASE_PASSWORD` 环境变量的值：

```bash
echo $DATABASE_PASSWORD
```

你应该看到从 `my-secret` Secret 检索到的数据库密码。
