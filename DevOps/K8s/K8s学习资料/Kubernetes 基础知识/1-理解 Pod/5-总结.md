# 总结

在本实验中，你掌握了 **Pod** 的概念，它是 Kubernetes 的基本构建块。

你成功完成了以下操作：

1. 为 Nginx Pod 编写了 **YAML 蓝图**。
2. 使用 `kubectl apply` 将其部署到集群。
3. 使用 `kubectl get` 验证其处于 **Running** 状态。
4. 使用 `kubectl describe` 查看了其启动历史记录。

理解 Pod 至关重要，因为 Kubernetes 中的所有其他概念——如 Deployments、Services、StatefulSets——最终都是为了管理这些 Pod。你现在已经准备好应对更高级的主题了！
