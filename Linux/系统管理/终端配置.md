## 安装 zsh 以及其他软件

```sh
sudo dnf install -y zsh curl git fzf
```

更换为 `zsh`

```sh
chsh -s $(which zsh)
```

## zimfw 框架

### 安装

[Home | Zim Framework: Modular, customizable, and blazing fast Zsh framework](https://zimfw.sh/)

安装脚本

```sh
# with curl
curl -fsSL https://raw.githubusercontent.com/zimfw/install/master/install.zsh | zsh

# with wget
wget -nv -O - https://raw.githubusercontent.com/zimfw/install/master/install.zsh | zsh
```


### 配置

配置完成后执行 `zimfw install` 生效

```diff
49c49,50
< zmodule zsh-users/zsh-syntax-highlighting
---
> # zmodule zsh-users/zsh-syntax-highlighting
> zmodule zdharma-continuum/fast-syntax-highlighting
55a57,60
>
> zmodule bira
> zmodule pabloariasal/zfm
> zmodule Aloxaf/fzf-tab
```

#### bira 主题

```
zmodule bira
```

#### 语法高亮插件

```sh
# 注释掉原来的插件
# zmodule zsh-users/zsh-syntax-highlighting

# 添加新的插件
zmodule zdharma-continuum/fast-syntax-highlighting
```

#### fzf-tab

tab自动补全时用 `fzf` 进行搜索

```
zmodule Aloxaf/fzf-tab
```

```sh
# 输出配置文件
fzf --zsh > ~/.config/zsh/completions/_fzf

应用
echo "zmodule ~/.config/zsh/completions --fpath ." >> .zimrc
```

#### 书签管理器

```
zmodule pabloariasal/zfm
```
