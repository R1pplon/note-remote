# .git泄露

## 操作

使用**GitHack**

```bash
python2 GitHack.py http://wwww.example.com/.git/
```

githack将自动下载并提取Git存储库的完整历史记录和配置信息，保存在当前目录下的.git目录中，进入该目录查看即可

```bash
# Ctrl+h 显示隐藏文件夹

grep -ri ctf *
# 在当前目录及其子目录下递归搜索包含"ctf"的文件或行。

git log
# 查看历史记录

git diff <commit>
# 查看某次提交

git strach list
# 列出所有暂存的更改记录

git strash pop
# 应用最近一次暂存的更改，并从stash列表中移除
```
